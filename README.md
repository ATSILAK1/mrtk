📌 Datacenter AR Visualization – Unity & WCF

This project is an Augmented Reality (AR) application developed using Unity 3D, designed to visualize datacenter infrastructure in an interactive and immersive way.

The application connects to a WCF backend service, which retrieves real-time data from a database containing detailed information about datacenter equipment (servers, racks, network devices, etc.). These data are then dynamically rendered into a 3D environment, allowing users to explore and interact with the datacenter layout in AR.

🚀 Features
📡 Integration with a WCF backend for data retrieval
🗄️ Dynamic loading of datacenter equipment from a database
🧊 3D visualization of racks, servers, and network components
🕶️ Augmented Reality interaction using Unity
⚡ Real-time representation of infrastructure data
🛠️ Technologies Used
Unity 3D (C#)
Augmented Reality (AR)
WCF (Windows Communication Foundation)
Database (SQL Server / MySQL depending on your setup)
🎯 Purpose

This project aims to improve the way datacenters are monitored and managed by providing a more intuitive and visual approach compared to traditional interfaces.

📌 Visualisation de Datacenter en Réalité Augmentée – Unity & WCF

Ce projet est une application en réalité augmentée (AR) développée avec Unity 3D, permettant de visualiser l’infrastructure d’un datacenter de manière interactive et immersive.

L’application est connectée à un backend basé sur WCF (Windows Communication Foundation), qui récupère les données depuis une base de données contenant des informations détaillées sur les équipements du datacenter (serveurs, racks, équipements réseau, etc.). Ces données sont ensuite utilisées pour générer dynamiquement une représentation 3D du datacenter en réalité augmentée.

🚀 Fonctionnalités
📡 Connexion à un service WCF pour la récupération des données
🗄️ Chargement dynamique des équipements depuis la base de données
🧊 Visualisation 3D des racks, serveurs et équipements réseau
🕶️ Interaction en réalité augmentée
⚡ Représentation en temps réel des données du datacenter
🛠️ Technologies utilisées
Unity 3D (C#)
Réalité Augmentée (AR)
WCF (Windows Communication Foundation)
Base de données (SQL Server / MySQL)
🎯 Objectif

Ce projet a pour objectif d’améliorer la gestion et la supervision des datacenters en proposant une approche plus visuelle, intuitive et immersive que les outils traditionnels.
