📌 Datacenter AR Visualization – Unity, WCF & HoloLens

This project is an Augmented Reality (AR) application developed using Unity 3D, designed to run on Microsoft HoloLens. It enables interactive and immersive visualization of datacenter infrastructure.

The application leverages the MRTK (Mixed Reality Toolkit) to provide natural interactions such as hand gestures, gaze, and object manipulation, ensuring a smooth mixed reality experience.

It is connected to a WCF (Windows Communication Foundation) backend service, which retrieves data from a database containing detailed information about datacenter equipment (servers, racks, network devices, etc.). This data is dynamically used to generate a 3D representation of the datacenter in AR.

- 🚀 Features
	- 🕶️ Compatible with Microsoft HoloLens
	- ✋ Natural interaction using MRTK (gestures, gaze, manipulation)
	- 📡 Integration with a WCF backend for data retrieval
  -🗄️ Dynamic loading of datacenter equipment from a database
	- 🧊 3D visualization of racks, servers, and network components
	- ⚡ Real-time representation of infrastructure data
- 🛠️ Technologies Used
	- Unity 3D (C#)
	- MRTK (Mixed Reality Toolkit)
	- Microsoft HoloLens
	- WCF (Windows Communication Foundation)
	- Database (SQL Server / MySQL)

🎯 Purpose
This project aims to modernize datacenter management and monitoring by providing an immersive and intuitive mixed reality solution, improving infrastructure understanding and operational efficiency.



// FRANCAIS //


📌 Visualisation de Datacenter en Réalité Augmentée – Unity, WCF & HoloLens

Ce projet est une application en réalité augmentée (AR) développée avec Unity 3D, conçue pour fonctionner sur le casque Microsoft HoloLens. Il permet de visualiser l’infrastructure d’un datacenter de manière interactive et immersive.

L’application utilise la bibliothèque MRTK (Mixed Reality Toolkit) pour gérer les interactions naturelles (gestes, regard, manipulation d’objets) et offrir une expérience utilisateur fluide en réalité mixte.

Elle est connectée à un backend basé sur WCF (Windows Communication Foundation), qui récupère les données depuis une base de données contenant des informations détaillées sur les équipements du datacenter (serveurs, racks, équipements réseau, etc.). Ces données sont ensuite utilisées pour générer dynamiquement une représentation 3D du datacenter en réalité augmentée.

- 🚀 Fonctionnalités
	- 🕶️ Compatible avec Microsoft HoloLens
	- ✋ Interaction naturelle via MRTK (gestes, regard, manipulation)
	- 📡 Connexion à un service WCF pour la récupération des données
	- 🗄️ Chargement dynamique des équipements depuis la base de données
	- 🧊 Visualisation 3D des racks, serveurs et équipements réseau
	- ⚡ Représentation en temps réel des données du datacenter
- 🛠️ Technologies utilisées
	- Unity 3D (C#)
	- MRTK (Mixed Reality Toolkit)
	- Microsoft HoloLens
	- WCF (Windows Communication Foundation)
	- Base de données (SQL Server / MySQL)
🎯 Objectif

Ce projet vise à moderniser la gestion et la supervision des datacenters en proposant une solution immersive en réalité mixte, facilitant la compréhension de l’infrastructure et améliorant l’efficacité des interventions techniques.
